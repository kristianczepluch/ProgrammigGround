Was ist eine Berechnung?
Aus entgegengenommenen Eingaben werden durch Berechnungen neue Werte generiert, welche an den Benutzer ausgegeben werden
oder an andere Funktionen als Zwischenergebnise weitergegeben werden.

Was verstehen Wir unter den Eingaben und Ausgaben für eine Berechnung? Nennen Sie Beispiele!
Eine Eingabe kann ein Wert sein den der Benutzer eingibt, oder eine Aktion wie ein Tastendruck, der aber lediglich 
auch nur einen Wert an uns weiter gibt. Dieser wird durch Berechnungen Verändert und als Ausgabe in Form eines Wertes
oder einer entsprechenden Aktion an den Benutzer (oder falls Zwischenergebnis an Funktionen) weiter/ausgegeben.

Was macht ein Ausdruck?
Ein Ausdruck kann eine Berechnung oder eine Variable sein.

Was ist der Unterschied zwischen einem Audruck und einer Anweisung?
Eine Anweisung ist ein Konstrukt, welches eine spezielle Aufgabe für uns erledigt, wie etwa eine Schleife oder eine
Ausgabe auf dem Bildschirm. Ein Anweisung kann ein Ausdruck sein, jedoch ist nicht jeder Ausdruck eine Anweisung.
(z.B eine Variable)

Was ist ein L-Wert? Nennen sie Operatioren, die einen L-Wert erfodern. Warum erfordern nur diese und nicht die anderen 
Opeartoren einen L-Wert?
Bei einem Ausdruck wie int zahl = 1 kann man vereinfacht sagen der L-Wert (links) ist der Speicher names zahl und der
R-Wert(rechts) ist der tatsächliche Wert, welcher innerhalb dieses Speichers liegt. Wird eine Berechnung vorgenommen,
so muss das Ergebnis meistens irgendwo gespeichert werden, folglich in einem L-Wert.

Was ist ein konstanter Ausdruck?
Ein Konstanter Ausdruck wird mit const gekennzeichnet und enthält einen Wert, der sich nie verändern soll, wie z.B Pi.
const double pi = 3.14159.

Was ist ein Literal?
Ein Literal wäre es wenn wir statt einer konstanten pi etwa eine Zahl verwänden würden, also ein Literal statt eines
Ausdrucks.

Was ist eine symbolische Konstante und warum verwenden wir sie?
Falls wir eine bestimmte Genauigkeit einer Variable brauchen, müssten wir die symbolische konstant etwa "const pi.."
lediglich verändern und müssten nicht den gesamten Code nach dieser Variable durchsuchen um sie einzelnd zu verändern.

Was versteht man unter magischen Konstanten?
Konstanten welche nicht mit const initialisiert wurden und deren Bedetung nicht klar ersichtlich ist.

Welche Operatoren können für Gleitkommazahlen aber nicht für ganze Zahlen verwendet werden?
Man kann bei Integer keine komplette Division durchführen, da hier immer der Rest ausgegeben wird.
(beeinhaltet auch multiplikation mit Faktoren < 0)

Welche Operationen können für ganze Zahlen aber nicht für Gleitkomma verwendet werden?
Mit Gleitkommazahlen lässt sich keine modulare Arithmetik durchführen.

Welche Operationen können für string objekte verwendet werden?
<,>,==,=! => kleiner alphabetisch, größer alphabetisch, identisch, ungleich.

Wann würde ein Programmierer eine switch Anweisung einer if Anweisung vorziehen?
Wenn man string Werte überprüfen muss, kann man nicht mit einer switch Anweisung arbeiten und muss auf eine if-else
Anweisung zugreifen. Prüft man Integer, Double oder Aufzählwerte so bietet sich ein Switch ausdruck an.

Welche Probleme treten oft bei switch Anweisungen auf?
Break könnte vergessen werden, in den case fällen dürfen keine Variablen verwendet werden und man kann nicht den gleichen
Wert für zwei Case Marke verwenden.

Welche Funktionen haben die einzelnen Teile im Kopf der for-Schleife und in welcher Reihenfolge werden sie verwendet?
Der erste Ausdruck "int i=0" steht für den Startwert, "i</x" steht für die Bedingung für eine Iteration und "++i" 
inkremetiert i, sodass die Schleife auch zu eine Zustand kommt, in dem die Bedingung nicht mehr erfüllt wird.

Wann sollte eine while und wann eine for Schleife verwendet werden?
Kann eine Schleife durch Initialisierung, Bedingung und Inkrementation erstellt werden sollte immer eine for-Schleife
verwendet werden. Ist dies nicht der fall (z.b immer wahr/unendlich Schleife) so sollte while verwändet werden.

Typumwandlung von char in int.

Beschreiben sie was die Zeile : char foo(int x) in einer Funktionsdeklaration bedeutet:
char drückt aus das diese Funktion einen character zurückibt, foo ist der Name der Funktion und (int x) die Argumenten-
liste, also welches Argument diese Funktion benötigt, um ausgeführt werden zu können.

Wann sollten Sie für einen Teil eines Programms eine seperation Funktion definieren? Nennen Sie Gründe?
Zum einen dienen Funktionen der Übersichtlichkeit und somit lassen sich speziell größere Berechnungen/Probleme in kleinere
Zerlegen. Es ist besonders Sinnvoll Funktionen für Berechnungen zu erstellen, welche häufig im Code benötigt werden.

Was können Sie mit einem int-Wert machen, das mit einem string-Wert nicht möglich ist?
Vergleiche mit Konstanten können bei ints mit switch erstellt werden, und mit string wird if-else benötigt.
Man kann einen int in einen char oder double konvertieren und berechnungen durchführen.

Was können Sie mit einem string-Wert machen, das mit einem int-Wert nicht möglich ist?
Man kann Zeichenkette speichern und diese durch die Operatoren <,> alphabetisch ordnen.

Wie lautet der Index des dritten Elements eines Vektors?
2, da es bei 0 beginnt also 0,1,2

Wie schreiben sie eine for-Schleife die alle Elemente eines Vektors ausgibt?
for(int i=0; i<vektorname.size();++i)

Was bewirkt vector<char>alphabet(26);?
Ein Vektor vom typ char mit dem Namen alphabet wird angelegt der 26 Werte Speicher und diese mit dem Standartwert eines
Char initialisiert.

Beschreiben Sie, wie push_back() einen vector verändert?
Mit dieser Funktion kann ein Wert an das Ende eines Vektors gehangen werden. (Es wird keine fixe Grö0e benötigt)

Welche Aufgaben haben die vector-Memberfunktionen begin(), end(), und size()?
begin() gibt den Anfang des Vektors an, end() das Ende und size die Anzahl der Elemente.

Weshalb sind Vektoren so beliebt/nützlich?
In ihnen können große Datenmengen kompakt gespeichert und durchiteriert werden. Außerdem muss keine fixe Größe angegeben 
werden.
